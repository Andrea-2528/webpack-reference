# Quick Start

## Getting Started

### 1. Clone the repository:
`git clone <repository-url>`

`cd <repository-folder>`

### 2. Install dependencies:

`npm install`

### 3. Start the development server:

`npm run dev`

### 4. Build for production:

`npm run build`
